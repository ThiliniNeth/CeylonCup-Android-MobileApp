# CeylonCup – Sri Lankan Tea M-Commerce Mobile App ☕

CeylonCup is a mobile commerce application designed to make purchasing authentic Sri Lankan tea simple and convenient. The app connects tea producers directly with consumers, providing a secure and user-friendly mobile shopping experience.

It addresses common challenges such as limited access to genuine tea products and the absence of specialized platforms focused on Sri Lankan tea.

## Overview 📱

CeylonCup offers a mobile-first platform where users can explore, search, and purchase premium Ceylon tea from anywhere. The application focuses on authenticity, convenience, and secure digital transactions while enabling wider access to Sri Lankan tea products. 

## Key Features

• Secure user authentication

• Product search and filtering

• PayHere payment gateway integration 

• Save products for future purchases (wishlist)

• Real-time order tracking

• Detailed product descriptions and reviews


## Technology Stack ⚙️

• Android Studio (Java)

• Firebase Firestore

• Firebase Authentication

• PayHere Payment Gateway

• Image upload support
